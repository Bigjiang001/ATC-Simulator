## v1.1.1 更新说明（2026-07-19）

### 今日升级汇总

- **管制指令**：增加英文语音速度指令，支持 `reduce/maintain/increase speed ... knots`，飞机会平滑调整速度；继续支持航向、高度、起飞、落地、复飞、离场程序和导航台移交。
- **跑道安全**：起飞前检查同跑道、反向跑道和交叉跑道占用；进港飞机到达入口时如跑道仍被占用，会自动复飞并播报原因。
- **飞行规则**：明确区分进港与出港航班，离场程序和导航台移交只允许出港航班使用；新进港航班从安全入口进入，减少穿越限制区的不合理航迹。
- **仿真时钟**：统一 `0.5x / 1x / 1.5x / 2x` 下的飞机运动、交通生成、繁忙阶段、进港预告和地面等待计时，暂停后所有仿真倒计时同步冻结。
- **间隔与碰撞**：碰撞和危险接近同时考虑水平、垂直间隔，高度充分分离的交叉航迹不再误判；真实相撞仍会立即结束游戏。
- **性能与稳定性**：修复高流量或低性能设备上部分飞机停止更新，以及暂停或结束后待进入航班仍继续倒计时的问题。
- **移动设备**：继续完善手机和 iPad 的触摸指挥、顶部设置布局、本地女声陆空通话和无线电效果。
- **地图验证**：Capital、Metro、Mountain 三张地图均已回归测试起飞、进近、落地、复飞、计分、移交、限制区与碰撞结束流程。

详细改动请查看 [CHANGELOG.md](CHANGELOG.md)。

其他改进：雷达标牌显示实时高度、速度和航向；支持按住左 Shift 发出语音指令；增加 `0.5x / 1x / 1.5x / 2x` 仿真速度。

欢迎来到空中交通管制模拟器，这是一款具有挑战性和沉浸感的游戏，让你化身为空中交通管制员。你的任务是管理空域，确保空中交通的安全和高效流动。

游戏玩法：
鼠标拖拽指挥：使用鼠标拖拽的方式来指挥飞机的航向。将等待起飞的飞机拖拽到跑道端口即可发出起飞指令；将即将落地的飞机拖拽到跑道端口处即可发出落地指令。

游戏规则：
安全间隔：如果飞机之间的间隔小于安全距离，游戏将结束。
管制区限制：飞机飞出管制区将导致游戏结束。
地面等待：地面等待的飞机超过1分钟，游戏也会结束。

主要特点：
互动控制：通过直观的控制按钮来开始、暂停和结束游戏。根据你的技能水平调整难度，从初学者到专家不等。
速度管理：轻松控制模拟速度。使用速度控制来减慢或加快游戏节奏，提供动态的游戏体验。
逼真的雷达界面：在详细的雷达屏幕上监控飞机的移动。运用你的技能安全地引导飞机通过空域。
语音指令：按住左 Shift 说出标准管制指令，松开后自动提交并由飞机执行；支持航向、高度、速度、起飞、落地、复飞、离场程序和导航台移交。
通信记录：通过可滚动的日志记录所有通信，确保你不会错过任何重要信息。
这款模拟器旨在测试你的多任务处理能力和在压力下的决策能力。无论你是初学者还是经验丰富的玩家，空中交通管制模拟器都能为你提供逼真且引人入胜的体验，让你全神贯注。

## 本地运行步骤

1. **克隆仓库**：
   ```bash
   git clone https://github.com/Bigjiang001/ATC-Simulator.git
   ```

2. **进入项目目录**：
   ```bash
   cd ATC-Simulator
   ```

3. **安装依赖**：
   ```bash
   npm install
   ```

4. **启动开发服务器**：
   ```bash
   npm run dev
   ```

5. **打开浏览器**：
   - 在浏览器中访问 [http://localhost:5173/](http://localhost:5173/) 查看项目。
